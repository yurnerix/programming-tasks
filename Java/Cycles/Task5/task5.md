# Задача 5. Эхо-ввод до слова `enough`

## Условие

С клавиатуры вводятся строки произвольного содержания.  
Необходимо выводить каждую введённую строку на экран до тех пор,  
пока не будет введено слово `enough`.  
Слово `enough` выводить не нужно — при его вводе программа должна завершить работу.

Рекомендуется использовать бесконечный цикл `while (true)` и оператор `break`.

## Входные данные

Последовательность строк.  
Ввод завершается, когда очередная строка ровно равна `enough`.

## Выходные данные

Все введённые строки, кроме слова `enough`, — каждая на отдельной строке, в том же порядке, что и на входе.
