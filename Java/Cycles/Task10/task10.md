# Задача 10. Статистика потока чисел

## Условие

С клавиатуры вводится поток строк. Каждая строка либо содержит целое число,  
либо является словом `stop`, которое завершает ввод.

Необходимо считать все числа до появления строки `stop` и вывести следующую статистику:

- количество чисел;
- сумму всех чисел;
- среднее арифметическое (вещественное значение);
- минимальное число;
- максимальное число;
- количество чётных чисел;
- количество нечётных чисел.

Если ни одного числа не было введено, для `min` и `max` можно вывести `N/A`.

## Входные данные

Последовательность строк.  
Ввод завершается строкой `stop`.

## Выходные данные

Несколько строк в формате:

count=КОЛИЧЕСТВО

sum=СУММА

avg=СРЕДНЕЕ

min=МИНИМУМ_ИЛИ_N/A

max=МАКСИМУМ_ИЛИ_N/A

even=КОЛИЧЕСТВО_ЧЁТНЫХ

odd=КОЛИЧЕСТВО_НЕЧЁТНЫХ
