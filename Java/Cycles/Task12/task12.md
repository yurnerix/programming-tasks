# Задача 12. Проверка магического квадрата

## Условие

Магическим квадратом называется квадратная матрица, в которой суммы элементов  
во всех строках, во всех столбцах и на обеих диагоналях совпадают.

Дана квадратная матрица размера `N × N`. Необходимо проверить, является ли она магическим квадратом.

## Входные данные

Сначала вводится целое число:
- `N` — размер матрицы (`N ≥ 1`).

Затем вводятся `N` строк, в каждой строке — по `N` целых чисел — элементы матрицы.

## Выходные данные

- `YES`, если матрица является магическим квадратом;
- `NO` — в противном случае.

## Пример

Ввод:

3

8 1 6

3 5 7

4 9 2

Вывод:

YES
