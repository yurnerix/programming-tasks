# Задача 11. Поиск подстроки (алгоритм Кнута–Морриса–Пратта)

## Условие

Даны две строки: текст `text` и шаблон `pattern`.  
Требуется найти все вхождения строки `pattern` в строку `text` и вывести  
начальные позиции этих вхождений (нумерация с 0).

Для эффективного поиска необходимо использовать алгоритм Кнута–Морриса–Пратта (КМП).

## Входные данные

Две строки:
1. `text` — произвольная строка (может содержать пробелы).
2. `pattern` — строка-шаблон, которую нужно найти в `text`.

## Выходные данные

Одна строка с целыми числами — индексами начальных позиций всех вхождений `pattern` в `text`,  
разделёнными пробелами.  

Если совпадений нет, допускается вывод пустой строки.

## Пример

Ввод:

ababaabababa

aba

Вывод:

0 2 7 9
