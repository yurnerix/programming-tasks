# Задача 2. Перебор кода замка

## Условие

Считается, что у замка есть правильный код — число 5.  
Программа должна перебрать все возможные коды от 1 до 10 включительно.  
Как только найден правильный код (число 5), необходимо вывести сообщение  
`Код найден! Замок открыт` и немедленно остановить перебор.

## Входные данные

Ввод отсутствует. Диапазон значений (от 1 до 10) задаётся в коде программы.

## Выходные данные

Одна строка:

Код найден! Замок открыт
