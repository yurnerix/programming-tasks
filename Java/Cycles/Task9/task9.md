# Задача 9. Простые числа до N (решето Эратосфена)

## Условие

По заданному числу `N` требуется найти все простые числа, не превышающие `N`,  
используя алгоритм решета Эратосфена.

Простое число — это число больше 1, имеющее ровно два различных натуральных делителя: 1 и само себя.

## Входные данные

Одно целое число:
- `N` (`N ≥ 2`).

## Выходные данные

- В первой строке — одно целое число: количество простых чисел, не превосходящих `N`.
- Во второй строке — сами простые числа в порядке возрастания, разделённые пробелами.

## Пример

Ввод:

10

Вывод:

4

2 3 5 7
