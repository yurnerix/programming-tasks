# Задача 6. Второе минимальное число

## Условие

С клавиатуры вводится последовательность целых чисел.  
Ввод продолжается до тех пор, пока вместо числа не будет введено что-то другое (например, слово `exit`).  
Необходимо определить второе по величине минимальное число среди всех введённых целых чисел и вывести его.

Если минимальных значений несколько, допускается вывод любого корректного второго минимального.  
Гарантируется, что на тестах будет введено не менее двух различных чисел.

## Входные данные

Последовательность строк. Каждая строка либо содержит целое число, либо значение,  
которое не может быть считано как целое (строка завершает ввод).

## Выходные данные

Одно целое число — второе минимальное среди введённых чисел.

## Пример

Ввод:

8

4

7

4

9

5

exit

Вывод (один из вариантов):

5
