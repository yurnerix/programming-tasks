# Задача 4. Контур прямоугольника

## Условие

Требуется вывести на экран прямоугольник высотой 10 строк и шириной 20 символов.  
Прямоугольник должен состоять из символа `Б`, при этом:

- первая и последняя строки полностью заполнены `Б`,
- в остальных строках символ `Б` должен стоять только в первом и последнем столбце,
- внутренность прямоугольника должна быть заполнена пробелами.

Реализация должна использовать вложенные циклы `while`.

## Входные данные

Ввод отсутствует. Размер прямоугольника (10 × 20) фиксирован в программе.

## Выходные данные

10 строк, изображающих контур прямоугольника из символов `Б`.
