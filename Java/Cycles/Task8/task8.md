# Задача 8. Спиральная матрица

## Условие

Дано целое число `N`. Необходимо сформировать квадратную матрицу размера `N × N`,  
заполненную числами от `1` до `N²` по спирали по часовой стрелке,  
начиная с верхнего левого угла.

После заполнения матрицы нужно вывести её на экран.

## Входные данные

Одно целое число:
- `N` — размер матрицы (`1 ≤ N ≤ 20`).

## Выходные данные

`N` строк, в каждой из которых содержится по `N` целых чисел, разделённых пробелами — элементы матрицы, заполненной по спирали.

## Пример

Ввод:

3

Вывод:

1 2 3

8 9 4

7 6 5
