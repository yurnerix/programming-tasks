# Задача 3. Заполненный прямоугольник из символов

## Условие

Необходимо вывести на экран прямоугольник высотой 5 строк и шириной 10 символов,  
полностью заполненный символом `Q`.  
Реализация должна использовать вложенные циклы `while` (цикл в цикле).

## Входные данные

Ввод отсутствует. Размер прямоугольника (5 × 10) фиксирован в программе.

## Выходные данные

5 строк по 10 символов каждая — прямоугольник из символов `Q`.

Пример вывода:

QQQQQQQQQQ

QQQQQQQQQQ

QQQQQQQQQQ

QQQQQQQQQQ

QQQQQQQQQQ
