# Задача 1. Суммирование чисел до стоп-слова

## Условие

С клавиатуры вводится последовательность целых чисел, по одному числу в строке.  
Ввод продолжается до тех пор, пока вместо числа не будет введено стоп-слово `ENTER`.  
Необходимо посчитать сумму всех введённых чисел **до** строки `ENTER` и вывести результат.

## Входные данные

Последовательность строк.  
Каждая строка до строки `ENTER` содержит одно целое число.  
Строка `ENTER` завершает ввод и числом не является.

## Выходные данные

Одно целое число — сумма всех введённых чисел до появления строки `ENTER`.

## Пример

Ввод:

14

11

19

9

ENTER

Вывод:

53
