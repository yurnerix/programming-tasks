# Задача 9. Медиана массива 

## Описание
Дан массив оценок.  
Отсортировать его и вывести медиану:  
- если длинна нечётная — центральный элемент  
- если чётная — среднее двух центральных

## Входные данные
Массив задан в коде.

## Выходные данные
Одно число — медиана.

## Требования
- Использовать `Arrays.sort()`.
- Корректно рассчитать медиану.

## Пример

Ввод: 7 10 4 8 6

Вывод: 7
