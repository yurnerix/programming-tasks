# Задача 1. Поиск самой высокой температуры

## Описание
Есть массив температур за день: `3, 11, 7, 4, 15, 2`.  
Нужно найти самое большое значение и вывести его на экран.

## Входные данные
Нет (массив задан в коде).

## Выходные данные
Одно число — максимальная температура.

## Требования
- Использовать массив типа `int[]`.
- Найти максимальное значение.
- Вывести результат в консоль.

## Пример
**Вывод:**  

15
