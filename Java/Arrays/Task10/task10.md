# Задача 10. Игровое поле 

## Описание
Создать поле 10×10 из символов.  
Заполнить его точками '.', затем вручную добавить несколько препятствий '#'.

## Входные данные
Нет.

## Выходные данные
10 строк по 10 символов.

## Требования
- Использовать `Arrays.fill()`.
- Вывести поле на экран.

## Пример

..........

..........

...#......

..........

....#.....

..........

..........

.#........

..........
..........
