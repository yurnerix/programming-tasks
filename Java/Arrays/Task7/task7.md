
# Задача 7. Максимальная температура по зонам

## Описание
Дан зубчатый массив температур по трём зонам.  
Нужно определить самую высокую температуру среди всех датчиков.

## Входные данные
Нет, массив задан в коде.

## Выходные данные
Строка:

Самая высокая температура: Х


## Требования
- Использовать зубчатый массив `int[][]`.
- Найти максимальное значение.
- Вывести сообщение.

## Пример
