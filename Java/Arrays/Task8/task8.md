# Задача 8. Вывод трёхмерного массива

## Описание
Есть трёхмерный массив `int[][][]`.  
Нужно вывести все значения, разделяя слои строкой `-----`.

## Входные данные
Нет.

## Выходные данные
Все числа в массиве, сгруппированные по слоям.

## Требования
- Пройти массив тремя циклами.
- Между слоями выводить `-----`.
