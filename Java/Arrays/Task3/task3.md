
# Задача 3. Сумма покупок

## Описание
Есть массив цен товаров: `5, 7, 2, 9`.  
Нужно вычислить общую сумму покупок.

## Входные данные
Нет.

## Выходные данные
Одно число — сумма элементов массива.

## Требования
- Создать массив `int[]`.
- Просуммировать значения.
- Вывести итоговую сумму.

## Пример

23
