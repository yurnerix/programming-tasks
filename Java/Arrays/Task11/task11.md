# Задача 11. Одинаковые ли массивы? 

## Описание
Есть два массива. Порядок элементов может отличаться.  
Нужно определить, содержат ли они одинаковое множество чисел  
(с тем же количеством повторов).

## Входные данные
Два массива заданы в коде.

## Выходные данные
`true` или `false`.

## Требования
- Копировать массивы: `Arrays.copyOf()`
- Отсортировать: `Arrays.sort()`
- Сравнить: `Arrays.equals()`

## Пример

a = 3 1 2 2

b = 2 3 2 1

Вывод: true
