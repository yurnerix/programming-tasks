# Задача 4. Нумерация мест в театре

## Описание
Зал имеет 2 ряда по 3 места.  
Нужно заполнить массив числами от 1 до 6 по строкам.  
Вывести место, расположенное во втором ряду и первом столбце.

## Входные данные
Нет.

## Выходные данные
Одно число — значение элемента [1][0].

## Требования
- Использовать массив `int[2][3]`.
- Заполнить числами по порядку.
- Вывести элемент во втором ряду, первом месте.

## Пример
**Вывод:**

4
