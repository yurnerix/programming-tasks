# Задача 2. Анализ массива вещественных чисел

## Описание

Дан одномерный массив из `N` вещественных чисел. Требуется:

1. Найти сумму отрицательных элементов.
2. Найти произведение элементов, расположенных между минимальным и максимальным значениями (не включая сами min и max).
3. Найти произведение элементов с чётными индексами (0, 2, 4, ...).
4. Найти сумму элементов между первым и последним отрицательными элементами (не включая их).

## Входные данные

- Целое число `N` (`N > 0`).
- `N` вещественных чисел.

## Выходные данные

4 строки:

1. `Сумма отрицательных: ...`
2. `Произведение между min и max: ...`
3. `Произведение элементов с чётными индексами: ...`
4. `Сумма между отрицательными: ...`

## Примечания

- Если между min и max нет элементов, произведение считается равным `1`.
- Если отрицательных элементов меньше двух, сумма между ними считается `0`.
