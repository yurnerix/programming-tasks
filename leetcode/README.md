# LeetCode 💡

В этой папке собраны мои решения задач с LeetCode на Java ☕

Здесь я практикуюсь в решении алгоритмических задач, изучаю структуры данных, разбираю разные подходы и учусь писать более эффективный и понятный код.

Основная цель этого раздела — развивать алгоритмическое мышление, лучше понимать популярные паттерны решения задач и постепенно накапливать собственную базу практики.

## Что находится в этой папке

Внутри каждой задачи обычно находятся:

- `README.md` — краткое описание задачи, идея решения, сложность и полезные заметки
- `Solution.java` — решение задачи на Java
